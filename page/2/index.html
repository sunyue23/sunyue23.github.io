创建版本库

初始化一个Git仓库，使用git init命令。

在Windows系统， 为了避免遇到各种莫名其妙的问题，请确保目录名（包括父目录）不包含中文

仓库建好后，可以发现当前目录下多了一个.git的目录，这个目录是Git来跟踪管理版本库的，没事千万不要手动修改这个目录里面的文件，不然改乱了，就把Git仓库给破坏了。

如果你没有看到.git目录，那是因为这个目录默认是隐藏的，用ls -ah命令就可以看见。

linux中date类型比较

#!/bin/bash

function get_interest_info()
{
echo -e “$1”
}

startTime=”$1”
endTime=”$2”
echo “startTime: $startTime”
echo “endTime: $endTime”
while [[ “$startTime” -ge “$endTime” ]]
do
get_interest_info $startTime
startTime=date -d "+1 day $startTime" +%Y-%m-%d
done
