安装JDK

  1. 下载：http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html

  2. 安装：默认安装即可，最终安装在C:\Program Files (x86)\Java目录下

  3. 环境变量配置：计算机→属性→高级系统设置→高级→环境变量

    • 系统变量→新建 JAVA_HOME 变量 。变量值填写jdk的安装目录

    • 系统变量→寻找 Path 变量→编辑。

      在变量值最后输入： %JAVA_HOME%\bin;%JAVA_HOME%\jre\bin;

      注意原来Path的变量值末尾有没有;号，如果没有，先输入；号再输入上面的代码

    • 系统变量→新建 CLASSPATH 变量

      变量值填写： .;%JAVA_HOME%\lib;%JAVA_HOME%\lib\tools.jar（注意最前面有一点）

  4. 检验是否配置成功 运行cmd 输入 java -version （java 和 -version 之间有空格）

    若如图所示 显示版本信息 则说明安装和配置成功。

安装Scala

  1. 下载：http://www.scala-lang.org/download/2.11.8.html

  2. 安装：直接将zip包解压 。

    需要注意的是最好不要将scala安装到带有空格的路径，否则可能在运行“scala”命令的时候

    报“此时不应有 \scala\bin\scala.bat)。”的错误提示。

  3. 环境变量配置：计算机→属性→高级系统设置→高级→环境变量

    • 系统变量→新建SCALA_HOME 变量 。变量值填写scala的安装目录

    • 系统变量→寻找 Path 变量→编辑。

      在变量值最后输入： %SCALA_HOME%\bin;%SCALA_HOME%\jre\bin;

      注意原来Path的变量值末尾有没有;号，如果没有，先输入；号再输入上面的代码

    • 系统变量→新建 CLASSPATH 变量（如果存在直接编辑，无需新建）

      变量值填写：

      .;%SCALA_HOME%\bin;%SCALA_HOME%\lib\dt.jar;%SCALA_HOME%\lib\tools.jar （注意最前面有一点）

  4. 检验是否配置成功 运行cmd 输入 scala。

安装hadoop

  1. 下载：http://www.apache.org/dyn/closer.cgi/hadoop/common/hadoop-2.6.5/hadoop-2.6.5.tar.gz
  2. 安装：将hadoop-2.6.5.tar.gz解压 。
  3. 环境变量配置：计算机→属性→高级系统设置→高级→环境变量
    • 系统变量→新建HADOOP_HOME 变量 。变量值填写hadoop的安装目录
  4. 安装hadoop插件：hadoop.dll和winutils.exe，将插件包放到HADOOP_HOME/bin目录下。

安装Maven

  2. 安装：直接将zip包解压 。

  3. 配置conf/settings.xml的localRepository

  4. 环境变量配置：计算机→属性→高级系统设置→高级→环境变量

    • 系统变量→新建MAVEN_HOME 变量 。变量值填写maven的安装目录

    • 系统变量→寻找 Path 变量→编辑。

      在变量值最后输入： %MAVEN_HOME%\bin;

      注意原来Path的变量值末尾有没有;号，如果没有，先输入；号再输入上面的代码
