配置选项

命令 示例 说明
daemonize daemonize yes 是否启用后台运行，默认no
port port 6379 设置端口号，默认6379
requirepass requirepass 123456 设置使用密码
databases databases 256 设置redis数据库总量
dir dir数据文件目录 设置数据文件存储目录
logfile logfile日志文件 设置日志文件

通用命令

命令 示例 说明
select select 0
select 1		 选择0号数据库
选择1号数据库
keys keys he*
keys he[h-1]*
keys ph?		 根据Pattern表达式查询复合条件的Key
注意：不要再生产环境中使用
dbsize dbsize 返回key的总数
exists exists a 检查key=a是否存在
del del a 删除key=a的数据
expire expire hello 20 设置key=hello20秒后过期
ttl ttl hello 查看key=a的过期剩余时间
flushdb flushdb 清空当前数据库下所有数据
flushall flushall 清空所有数据
